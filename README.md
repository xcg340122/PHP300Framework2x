* MVC开发框架

* 支持CLI模式运行
* 核心容器加载处理
* 自动JSON编码
* 自定义高可用扩展方法
* `Composer`随意`DIY`

* 数据库驱动快速操作
* 分离式入口更安全
* 高速缓存扩展支持
